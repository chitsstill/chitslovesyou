# Chit Heart Website

This project is a simple web application that displays a heart on a black background. When the heart is clicked, it dissolves to reveal the message "Chit loves you."

## Project Structure

```
chit-heart-website
├── src
│   ├── index.html       # Main HTML document
│   ├── css
│   │   └── styles.css   # Styles for the webpage
│   └── js
│       └── main.js      # JavaScript for handling interactions
├── .gitignore           # Git ignore file
└── README.md            # Project documentation
```

## Setup Instructions

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   cd chit-heart-website
   ```

2. **Open the project in your preferred code editor.**

3. **Run the project:**
   - Open `src/index.html` in a web browser to view the website.
   - You can use a live server extension in your code editor for a better development experience.

## Features

- A heart graphic that responds to user clicks.
- A smooth animation that reveals the message "Chit loves you" upon clicking the heart.

## Technologies Used

- HTML
- CSS
- JavaScript
